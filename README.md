# 3df2obj.c

This tool convert 3D mesh model from original [Moto Racer](https://en.wikipedia.org/wiki/Moto_Racer) game to .obj file.
Details of .3df file format in 3df2obj.c.

# Usage

> ` ./a.out  ./input_file.3df ./output_file.obj`
